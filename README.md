# 🌐 UsabilityHub Clone

A fully responsive front-end project inspired by the design of UsabilityHub. This project focuses on recreating a modern landing page using HTML and CSS.

---

## 📌 Features

- Fully responsive design (mobile, tablet, desktop)
- Clean and modern UI layout
- Flexbox-based layout system
- Reusable components (buttons, cards, sections)
- Hero section with call-to-action
- Feature cards and examples section
- Footer with multiple columns

---

## 🛠️ Built With

- HTML5  
- CSS3  
- Flexbox  
- Media Queries  

---

## 🎨 Design Reference

This project is inspired by the UsabilityHub design:

https://usability-hub-clone.vercel.app/#

---

## 📱 Responsive Design

The website is optimized for different screen sizes:

- Desktop (1200px and above)
- Tablet (768px – 1199px)
- Mobile (below 768px)

---

## 🎯 Purpose

This project was created to:

- Practice responsive web design  
- Improve layout skills using Flexbox  
- Understand real-world UI structure  
- Recreate a professional landing page design
